## 2. 取景
### 2.1 焦距
* 焦距越短，拍摄的视野越宽，取景范围越广，画面中容纳的景物越多，每个景物在画面中占的面积越小
* 焦距越长，拍摄的视野越窄，取景范围越窄，画面中容纳的景物越少，每个景物在画面中占的面积越大
{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/038dfebe9ec383873ac7b0e0d141d8d4.webp" caption="焦距图例" align="center" width=600px >}}
**等效焦距**

{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/47859f07342669e00badf8baa83df14b.webp" caption="画幅尺寸" align="center" width=600px >}}

公式：实际焦距 * 焦距转换系数 = 等效焦距
1) 全画幅相机只能用全画幅镜头。
2) 全画幅镜头既可以用在全画幅相机上，也可以用在半画幅相机（APS-C画幅相机的俗称）上。
3) 可换镜头上面标注的焦距都是实际焦距。全画幅镜头和半画幅专用镜头都是如此。
4) 只要是半画幅机身，不管用什么镜头，就一定要通过焦距转换系数求得等效焦距。
5) 全画幅系数是1，就是不需要计算，等效焦距就是实际焦距。尼康、索尼、宾得、富士半画幅相机的焦距转换系数是1.5。佳能半画幅相机的焦距转换系数是1.6。奥之心和松下的M/43系统是2。
### 2.2 透视
* 透视就是指近大远小，透视永远客观存在
* 拍摄距离越近，透视效果越明显；拍摄距离越远，透视效果越不明显，直到可能看不出来
### 2.3 如何取景
中心思想：更好的表达<mark style="background: #FFB8EBA6;">“主题”</mark>
* 取景范围：取景进画面的元素要尽量对主题有帮助，至少不能影响主题的表达
* 取景角度：帮助照片中的主体展现不同的状态，决定画面中有哪些元素
* 取景时机：记录精彩瞬间（可以借助连拍，视频等功能）
### 2.4 取景常见问题
拍摄时应该思考的问题
* 景色并不好看
* 主体处于不好的状态
* 拍摄的角度并不好
* 画面缺主体
* 取景范围不对
* 前景画蛇添足
* 背景中有不该有的东西
### 2.5 取景的摄影语言
让拍摄的照片“更进一步”
* 控制取景范围营造反差
* 不寻常的视角
* 让画面更有代入感
* 让画面更有冲击力
* 善于利用前景
## 3. 曝光
### 3.1 “白加黑减”
#### 3.1.1 影调
照片的明暗程度
* 中间调 - 18%灰（现实生活中所有物品的反射率）
* 高调 - 比中间调亮的
* 低调 - 比中间调暗的
{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/040f94d50c5c187eb932a01c0d28514d.webp" caption="18度灰" align="center" width=600px >}}
#### 3.1.2 测光
* 作用：相机衡量测光区域亮度的功能
* 测光模式：测光区域、计算权重分配不同
	* 点测光
	* 评价测光（矩阵测光、多重测光）<mark style="background: #FFB86CA6;">*√ 优选*</mark>
	* ...
* 相机开启后会一直测光
#### 3.1.3 曝光补偿
曝光值是用来量化一张照片明暗程度（影调）的参数
{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/b979b943b8fef3f78446c3728431b7b3.webp" caption="曝光补偿" align="center" width=600px >}}
* 曝光补偿标尺为0(EV) - 中间调
* 曝光补偿标尺为正值 - 高调，正值越大照片越明亮。每增加1EV，亮度 * 2
* 曝光补偿标尺为负值 - 低调，负值越大照片越昏暗。每减少1EV，亮度 / 2
#### 3.1.4 白加黑减
* 如果取景范围内很白，我们要拍出这样原本的白，需要增加曝光补偿（0EV会欠曝），是为“白加”
* 如果取景范围内很黑，我们要拍出这样原本的黑，需要减少曝光补偿（0EV会过曝），是为“黑减”
### 3.2 曝光三要素
{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/dc5019b516bdfd4a7d4db65ce85e207c.webp" caption="曝光三要素" align="center" width=600px >}}
#### 3.2.1 光圈
镜头里的一个可以变大变小的孔
{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/3f32bfdc813d8dd4dfca2065c79b8382.webp" caption="光圈" align="center" width=600px >}}

* 光圈值 = 焦距 / 光圈孔径 
	* 光圈值越小，光圈越大
	* 常见光圈值：f/1.0、f/1.4、f/2、f/2.8、f/4、f/5.6、f/8、f/11、f/16、f/22、f/32，每一档光圈值都是$\sqrt{2}$ 倍的关系
* 光圈决定了通光速率。光圈越大，单位时间内通过的光线越多，照片越明亮
* 光圈影响虚化效果。光圈越大，背景虚化能力越强
* 其他辅助效果：
	 * 小光圈可以拍出星芒效果
	 * 大光圈可以让前景消失
	 * 小光圈可以检验镜头清洁度
	 * 大光圈可以排除不同形状的光斑
#### 3.2.2 快门速度
{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/c7f049badd2958c26e6be888bfeb3763.webp" caption="快门速度" align="center" width=600px >}}

* 快门速度越高，曝光时间越短，相对来说曝光量越小
* 快门速度越低，曝光时间越长，相对来说曝光量越大
* 常见快门速度：30s、15s、8s、4s、2s、1s、1/2s、1/4s、1/8s、1/15s、1/30s、1/60s、1/125s、1/250s、1/500s、1/1000s、1/2000s、1/4000s、1/8000s，相邻两档快门速度之间都是2倍或约2倍的关系。快门速度每调低一档，曝光量就约变成原来的2倍
#### 3.2.3 感光度
{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/80b7fe495df1462a4242fd4a466f9a29.webp" caption="ISO" align="center" width=600px >}}

* 感光度（ISO）就是指感光元件对光线的敏感程度
* 常见ISO值：50、100、200、400、800、1600、3200、6400、12800、25600、51200等，相邻两档ISO值基本都是2倍的关系。ISO每提升一档，感光元件对光线的敏感程度就变为上一档的2倍
* ISO越高，CCD和CMOS越容易被电信号干扰，照片的噪点越多，看起来越粗糙（锐度下降 -> 细节损失 -> 噪点增加 -> 色彩偏移）
#### 3.2.4 互易律
光圈、快门速度、感光度，这三者其中一个或两个参数，朝明亮或昏暗的方向一共调节n档，这三者中的另外两个或一个参数，朝相反的方向也一共调节n档，那么最终曝光补偿，也就是拍的照片的亮度不变。

### 3.4 正确的曝光
主观的，需要符合拍摄意图以及要表达的照片主题
#### 3.4.1 主观感受
* 对整张照片的感觉
* 画面的主要元素是不是曝光准确
* 在主题已经成为剪影时，背景色彩好看很重要
#### 3.4.2 直方图
{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/505cee9613f17ccdadafad4e42f02f24.webp" caption="直方图" align="center" width=600px >}}

* 横轴表现画面从暗到亮的趋势，左边暗，右边亮
* 纵轴表示像素数量高低，越高像素数量越多
#### 3.4.3 大光比场景拍摄技巧
大光比、强反差的场景，就是亮的地方特别亮，暗的地方特别暗
* 白就白吧，黑就黑吧
* 提亮暗部，压暗亮部（补光、减光）
* 高动态范围HDR（包围曝光）
* 拉上拉链（光线不好的时候咱不拍）
### 3.5 光线
#### 3.5.1 光线的特性
* 光线的强度：与光源能量成正比，与光源距离的平方成反比
* 光线的方向
	* 顺光 - 光源在被摄主题正面
	* 侧光 - 光源在被摄主题正面45°的位置
	* 90°侧光 - 光源在被摄主题正面90°的位置
	* 逆光 - 光源在被摄主题背面
* 色温 - 光色的尺度
	* 蓝色是冷色，但是色温高
	* 红色是暖色，但是色温低
	* 白平衡 - 将在不同环境色温中的白色物体都还原成真的白色
#### 3.5.2 光线的分类
* 自然光
	* 以阳光作为光源
	* 日出之前和日落之后自然光的色温很高，颜色冷艳迷人，但是光线强度很低，有一种神秘的气息
	* 日出、日落时的光线色温很低，光线温暖、柔和
	* 晴天的日光色温居中，光线强度较高
	* 阴天和雨天，光线色温居中，但强度较低
	* 耶稣光，需要主体景物好看
* 场景光
	* 室内遇到的光线
* 人造光
	* 光线的强度、方向、色温全都由摄影师控制，为摄影服务
### 3.6 曝光常见问题
* 过曝或者欠曝
* 超过宽容度范围（出现死白或死黑）
* 非得逆光
* 室外强光是“杀手”
* 色温太偏了
### 3.7 曝光与情绪
* 通过影调来表达情绪（正加负减）
* 明暗的反差
* 色彩的渲染（冷暖色调）
	* 同类色 - 色相环中某一种颜色左右15°内的颜色，给人和谐、舒适的感受
	* 互补色 -  色相环中180°相对的两种颜色，给人强烈对比、激烈的感受
	{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/778c09b3d7e0fe714b951dc8ca50b073.webp" caption="色相环" align="center" width=600px >}}

* 曝光的“累积” - 相机给了您黑色的画布，摄影让您涂抹光明
## 4. 虚实
### 4.1 对焦与景深
#### 4.1.1 对焦
* 对上焦的地方一定是清楚的
* 没对上焦的地方不一定是模糊的
	* 焦平面是一个面，面上的内容都是清楚的
	* 存在景深
#### 4.1.2 景深
* 焦平面前面的叫前景深，后面的叫后景深
* 景深与人眼的分辨能力和照片的实际大小有关。离开了对焦点，画面就开始模糊，离得越远就越模糊。但如果离得很近，虽然会模糊，但由于照片尺寸不够大，或者人眼难以分辨，仍然会认为是实像
* 景深特点：
	* 景深和光圈、焦距、拍摄距离有关
	* 前景深比后景深要浅（短），前面的容易“虚”，后面的不容易“虚”
	* 随着拍摄距离、焦距、光圈等参数的变化，后景深变化特别快，前景深变化慢
	* 景深范围小的话，“实”的区域就小，这被我们称为浅景深。景深范围大的话，“实”的区域就大，这被我们称为深景深。
### 4.2 通过景深实现虚化
* 背景远：让背景处于景深之外，且越远越好，就可以实现虚化
* 相机近：相机离拍摄主体越近，景深就越浅，背景虚化效果就越明显
* 光圈大：小孔成像原理，小孔能成实像，孔大了会成虚像
* 焦距长：焦距越长背景虚化效果就越明显
### 4.3 快门控制虚实
画面中一定要有“动”的元素
* 利用高速快门“凝固”瞬间 - 时间点
* 慢速快门记录时间 - 时间段
### 4.4 虚实常见问题
* 没对上焦
* 对焦错了
* 余弦误差 - 先对焦再构图导致
* 景深太深（该虚化的没虚化）或者太浅（该拍清楚的没拍清楚）
* 手抖造成的虚 - 没达到安全快门
* 没有“凝固”画面主体
## 5. 构图
### 5.1 构图法
#### 5.1.1 居中构图法
将主体放在画面中间：
* 主体在画面中占比很大
* 拍对称的场景
#### 5.1.2 三分构图法
将画面从上到下分成3等份，从左到右也分为3等份。画面中的主线条放在三分线上，拍摄主体放在三分线的4个交点（视觉中心）上，会让看照片的人第一时间注意到主体和主线条，同时也会让人觉得画面看起来很舒服
{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/e6d0b6b25ad85651e7360403b7eeab09.webp" caption="三分构图法" align="center" width=600px >}}
*相对居中构图法，三分构图法强调的就是<mark style="background: #ABF7F7A6;">不居中</mark>，不要刻板使用，摄影无定法*
#### 5.1.3 重复法
拍摄的画面没有单个明确的主体，主体可能是一群相同的东西
* 单一元素
* 不断重复
* 使之尽量充满画面
#### 5.1.4 引导线法
当画面中的主要线条汇聚到一点时，会容易被人眼捕捉到。可以依靠线条对视线的汇聚作用来突出拍摄主体
#### 5.1.5 三角构图法
画面中的元素或者线条形成了三角形
* 正三角形 - 画面舒服、稳定
* 倒三角形 - 画面不稳定、倾斜
#### 5.1.6 框架构图法
画面中有元素构成天然的框架，人们会习惯于往框架中间去看。

### 5.2 摄影构图常识
#### 5.2.1 横构图和竖构图
* 横构图：给人稳定和宽阔的感觉，适合大场景风光
* 竖构图：表达画面的纵深，以及高低的感受
#### 5.2.2 构图的基本常识
* 点：
	* 点在画面中心会受力均衡，所以显得呆板
	* 点在画面上方会显得上浮
	* 点在画面下方会显得下沉
* 线条：
	* 横向的线条会让画面显得开阔
	* 纵向的线条会让画面显得高耸
	* 曲线会让画面显得柔和以及绵延
* 形状：
	* 正三角形会让画面显得稳定，倒三角形会让画面显得不稳，或者有冲击力
	* 方形常见于拍建筑物
	* 圆形在画面中出现的时候，画面会显得聚拢，有一种自然的收缩感
### 5.3 3.构图常见问题
* 太小的主体居中
* 水平线、地平线居中
* 该水平的线不水平 - 画面歪了
* 前面空间小、后面空间大 - 人物有朝向时，会让画面显得很”堵“
* 把人拍得太矮 - 人的脑袋放在中线及以下
* ”切人“（画面四条边切断人的身体）的位置不对 - 关节、手脚、头或脖子被画面切了
* 脑袋上”长树“
