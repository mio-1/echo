## 1. 乐音体系
1. **音的四种性质：** 高低，长短，强弱、音色。
2. **噪音：** 振动不规则，音的高低听起来不明显。
3. **乐音体系：** 音乐中所使用的基本的乐音的总和。
4. **音级：** 乐音体系中的各音（专指乐音）。
5. **音列：** 按一定的音高关系和高低次序，由低到高或由高到低排列。
6. **半音：** 乐音体系中，音高关系的最小计量单位。
7. **全音：** 半音和半音相加。
8. **音名：** C D E F G A B 也叫基本音级，一个音级到下一个的距离为“八度”，do re mi fa sol la si 这些音名多用于歌唱，故叫**唱名**。{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/1da147feb3b421d5ef6f69cd18e1be98.webp" caption="音名和唱名" align="center" width=600px >}}
9. **变化音级：** 将基本音级加以升高或降低得来的音。
10. **升级音级：** 将基本音级升高半音，如<sup>#</sup>C、<sup>#</sup>D、相反是**降级音级**。
11. **重升音级：** 将基本音级升高全音，如<sup>x</sup>C、<sup>x</sup>D等，反之为**重降音级**。
12. **音组：** 乐音体系中八十多个音用来区分音分为若干组，它的标记是用字母加数字“1”来表示。比小字一组高的各组，由低到高名为“小字二组”、“小字三组”、“小字四组”、“小字五组”。依次写上数字“2”、“3”、“4”、“5”。比小字一组低的各组，由高到低依次定名为“小字组”、“大字组”、“大字一组”、“大字二组”。小字组用小写字母，数字写在音名右上方，如c<sup>1</sup>。大字组用大写字母，数字写在音名右下方，如C<sub>1</sub>。{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/d7d03a18ef3153b165c514ae7185d692.webp" caption="音组" align="center" width=600px >}}
13. **音域：** 从低音到高音，音列的总范围。
14. **音区：** 音域中的一部分。根据音色的不同分为高音区、中音区和低音区。小字组、小字一组、小字二组被认为是中音区，小字三，小字四，五为高音区，大字组，大字一组和二组为低音区。
## 2. 音律
1. **标准音：** 目前国际通用的标准高度是以440Hz的小字一组的a为标准，因此a<sup>1</sup>就成了标准音。
2. **中央C：** 位于乐音体系中央的小字一组的c<sup>1</sup>。
3. **定律法：** 确定乐音体系中各音的绝对准确高度，人们在实践中创造了各种定律法。如十二平均律，五度相生律、纯律等。
4. **十二平均律：** 将一个纯八度（如c<sup>1</sup>-c<sup>2</sup>）分成十二个均等的部分。
5. **等音：** 音高相同而记法和意义不同的音。如<sup>#</sup>C、<sup>b</sup>D，<sup>x</sup>B，这三个音在钢琴上音高是完全相同的，但记法和意义不同。可以看出，除了<sup>#</sup>G和<sup>b</sup>A只有一个等音外，其他各音都有两个等音。{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/49f603e8236bd67d4381e62637e100b5.webp" caption="等音" align="center" width=600px >}}
6. **基音：** 由全弦振动产生的音，听的最清楚的最响的音。
7. **泛音：** 由发音体各部分振动而产生，不易被听出的音。
## 3.调式
1. **调式：** 若干音（3-7个）按照一定的关系（高低、稳定，不稳定）联结在一起，构成一个体系，并以某一个音为中心的体系。
2. **大调式：** 简称大调，是一种由七个音构成的一种调式，其稳定音合在一起构成一个大三和弦，不稳定音以二度音程关系倾向于稳定音，构成旋律进行的基本音调。根本特征表现为主音上方的大三度。分为三种：自然大调、和声大调和旋律大调。
	* 自然大调{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/9afb330ce150a1f57deeecef0b76e0fc.webp" caption="自然大调" align="center" width=600px >}}{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/c6508dfb9ca9633dbae7eb4aa83f3fe9.webp" caption="自然大调" align="center" width=600px >}}
	* 和声大调
	* 旋律大调
3. **音阶：** 调式中的音，从主音开始到主音结束，按高低次序排列起来。
4. **等音调：** 根据十二平均律所有半音相等的理论，不仅产生了等音、等音程、等和弦，还产生了等音调。就是两个调之间的所有音都是等音关系。
	* 黑键开始的等音调：{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/892e29b4c45557b4d57627d75ce3c052.webp" caption="等音调" align="center" width=600px >}}
	* 常用的15个大调{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/05d506eac9e52db856719a930ca4d37c.webp" caption="常用的15个大调" align="center" width=600px >}}
5. **首调：** 任何一个大调的首音的唱名都为do，其他音依次类推
6. **固定调：** 音名C D E F G A B固定对应唱名do re mi fa sol la si 
7. **原调、移调、转调、离调：** {{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/b597c2f577f27b25260b631d5480bd98.webp" caption="原调、移调、转调、离调" align="center" width=600px >}}
## 4. 节奏节拍
1.  **节拍：** 有强有弱的相同时间片断，按照一定的次序循环重复。
2. **节奏：** 用强弱组织起来的音的长短关系。
3. **拍子：** 节拍的*单位拍*，用固定的音符来代表，叫做“拍子”。
4. **拍号：** 表示拍子的记号。拍号用分数的形式标记。{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/e31bce8918eaeb58434302f02bc8f528.webp" caption="拍号中的强弱关系" align="center" width=600px >}}
5. **小节：** 在乐曲中，由一个强拍到次一强拍之间的部分。
6. **小节线：** 永远作为强拍的标记写在强拍的前面
7. **弱起小节：** 乐曲由弱拍或次强拍开始。是不完全小节，结尾一般也是不完全小节，两个不完全小节合在一起成为一个完全小节。{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/d10695ef8fa9675c9e058015731052da.webp" caption="弱起小节" align="center" width=600px >}}
8. **单拍子：** 每小节有两拍或三拍，只有一个强拍，并且也存在弱拍。
9. **复拍子：** 由完全相同的单拍子结合在一起。{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/baaca7841fdde8e3cf095c54528bef50.webp" caption="复拍子" align="center" width=600px >}}
10. **混合拍子：** 由不同的单拍子按照不同的次序结合。
11. **一拍子：** 每小节一拍，每拍都是强拍，没有弱拍。
12. **散拍子：** 拍子的强音位置和单位拍的时值都不是十分明显，也不固定。
13. **变换拍子：** 各种拍子交替出现（拍号至少发生过一次改变）。
14. **速度与情绪：**
	 * 具体的速度{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/bdf5effaba829fc3226e7bd0a74e6046.webp" caption="具体的速度" align="center" width=600px >}}
	 * 抽象的速度{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/e46261a518d00c011ee83043e775ca18.webp" caption="抽象的速度" align="center" width=600px >}}{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/892cd6f9aa08149e1b2553c49886da0f.webp" caption="抽象的速度" align="center" width=600px >}}
15. **切分：** 改变了拍的内部强弱关系。{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/f72771d915cadebc28fdb7eafd1f9f62.webp" caption="切分" align="center" width=600px >}}{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/a02861e6148964a6cdcff1a2e2773e35.webp" caption="image.png" align="center" width=600px >}}
## 5. 音程
1. **音程：** 在乐音体系中，两音之间的高低关系。{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/84d9db5d9d9bb16fd411bc14ef1226d3.webp" caption="" align="center" width=600px >}}{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/5c9f3d72eaad9c82bfcdf6408a5c5aa8.webp" caption="" align="center" width=600px >}}{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/caef4dea42a6e24376b8c956c92debf2.webp" caption="" align="center" width=600px >}}
 
	* 快速识别音程
		* 不涉及升降号，只看包含ef和bc数量 {{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/a62a6209aa9610a72e5975c258f909b2.webp" caption="" align="center" width=600px >}}
		* 通用判断方式  {{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/713b4592209892f022cbc2a16c7774a2.webp" caption="" align="center" width=600px >}}
2. **冠音、根音：** 音程中，高的音叫冠音，低的音叫根音。
3. **音程的度数：** 音程在五线谱上包括的线与间的数目。
4. **单音程：** 八度以内（包括八度）的音程。
5. **复音程：** 超过八度的音程。
6. **等音程：** 两个音程孤立听时，音响效果完全相同，但在乐曲中的意义和写法不同。
7. **旋律音程：** 音程中的两个音，先后发声。
8. **自然音程：** 纯音程、大音程、小音程、增四度和减五度。
9.  **变化音程：** 除增四减五外的一切增、减、倍增、倍减音程。
10. **协和音程：** 听起来悦耳、融合的音程。
11. **不协和音程：** 听起来不融合的音程。
12. **音程的转位：** 音程的根音和冠音相互颠倒。{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/765667c9e50302c1a5a7982979d44d38.webp" caption="" align="center" width=600px >}}
## 6. 和弦
1. **和弦：** 按照三度音程关系或非三度音程关系、三个以上的音的结合。
2. **三和弦：** 按照三度音程关系叠置起来的三个音所构成的和弦。音程排列的根音、三音、五音分别用1、3、5 来表示。 {{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/cb9f77a5cd743cb308c1ee1d1ee47187.webp" caption="" align="center" width=600px >}}{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/0f2428852e1a18bd612696954bc0a232.webp" caption="" align="center" width=600px >}}{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/e7e856aa63295d195b4b2ad3f08e5af8.webp" caption="" align="center" width=600px >}}
3. **七和弦：** 按照三度音程关系叠置起来的四个音所构成的和弦。用1、3、5、7来表示 。{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/0abbce115754bf25b27e816c9ffa8f35.webp" caption="" align="center" width=600px >}}{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/3d749b00e3719c2c4f0cac87aa0bbd18.webp" caption="" align="center" width=600px >}}{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/55ba5fac316e6dbfd44a1b3f9e7a2989.webp" caption="" align="center" width=600px >}}
4. **和弦固定标记法：** {{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/a806f1047a5d71b81ba305fbe23e352f.webp" caption="" align="center" width=600px >}}{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/f5f062c1f47e576cb8e9553f1a22f558.webp" caption="" align="center" width=600px >}}{{< fancybox_figure src="https://i-echo.oss-cn-shenzhen.aliyuncs.com/img/776548b5795179616eb5803933193cc3.webp" caption="" align="center" width=600px >}}
